# Proyecto NOTAS

Esta aplicación nos permite registrarnos y crear un usuario con un espacio que nos permite la creación de notas que pueden ser removidas y/o editadas. El proyecto fue creado con Node.JS utilizando express para la parte del servidor y como práctica para la creación de una API, base de datos para guardar las credenciales de distintos usuarios y una interfaz para para que los usuarios puedan realizar solicitudes al servidor.
## Tecnologías utilizadas

- **Node.JS**
- **Express**
- **JavaScript**
- **Bootstrap**
- **PostgreSQL**
## Despliegue del proyecto

El proyecto actualmente se encuentra desplegado utilzando Netlify para servir el Frontend y Render como servidor para nuestra API de igual manera en Render se aloja la base de datos para guardar los datos de los usuario registrados, como también las notas que cada usuario crea en un cuenta personal. La aplicación también cuenta con validación de tokens para verificar que la sesión sea válida y no poder ingresar a ver las notas hasta que se inicie sesión correctamente.
